# Workshop 3 · Module 3 — Homework
## Comparison Charts

**Due**: Before Workshop 3, Module 5
**Format**: R script + Python script + PDF report (max 4 pages)
**Weight**: Part of Workshop 3 homework (5% of total grade)

### Part A — Horizontal Sorted Bar (20 points)
Using Google Play Store data, produce a horizontal bar chart of the top 10 categories by count. Sort by value, add direct labels, apply grey-accent on the maximum. Produce in R and Python.

### Part B — Three Bar Variants (25 points)
Produce grouped, stacked, and 100% stacked bar charts of Content Rating × Type. Arrange as a 1×3 panel. In 100 words, explain which variant best answers: "What proportion of Mature apps are Paid?"

### Part C — Lollipop + Cleveland Dot (25 points)
Produce the same top-10-category data as (a) lollipop and (b) Cleveland dot plot. Place all three (bar, lollipop, dot) in a 1×3 panel. In 100 words, compare data-ink ratios and audience suitability.

### Part D — Dumbbell (30 points)
Create synthetic data for 6 categories with values in 2020 and 2024. Produce a dumbbell chart with: grey connector, blue (2020), red (2024), direct change labels (+N). Sort by 2024 value. Produce in R and Python. In 100 words, explain when dumbbells are preferred over grouped bars.
