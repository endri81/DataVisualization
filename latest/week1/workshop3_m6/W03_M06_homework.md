# Workshop 3 · Module 6 — Homework
## Rankings: Slope, Bump & Waterfall

**Due**: Before Workshop 3, Module 8
**Format**: R script + Python script + PDF report (max 4 pages)
**Weight**: Part of Workshop 3 homework (5% of total grade)

### Part A — Slopegraph (30 points)
Create synthetic data for 8 categories with values in 2020 and 2024. Produce a slopegraph with grey-accent on the biggest mover. Add direct labels at both ends. Produce in R and Python. In 100 words, explain when a slopegraph is better than a grouped bar chart.

### Part B — Bump Chart (35 points)
Using Google Play Store data, compute the rank of the top 6 categories by app count for 6 simulated years (2019–2024). Produce a bump chart with inverted y-axis, direct endpoint labels, and grey-accent on the top 2 categories. Produce in R and Python.

### Part C — Waterfall (35 points)
Create a P&L waterfall with: Revenue ($1,000), COGS (-$400), Gross Profit ($600), OpEx (-$250), EBITDA ($350), Tax (-$70), Net Income ($280). Use blue for totals, green for positives, red for negatives. Add connector lines and value labels. Produce in R and Python.
