# Workshop 3 · Module 10 — Course Notes
## Lab: Chart Selection Decision Framework

### 1. The Decision Framework
The chart selection flowchart asks one question: "What do you want to show?" The answer maps to five primary chart families: distribution (histogram, density, ridgeline, box, violin), comparison (bar, lollipop, dot, dumbbell), relationship (scatter, bubble, hexbin), proportion (pie, donut, treemap, waffle, stacked bar), and change/ranking (line, slope, bump, waterfall). Three enhancement layers apply to any chart: uncertainty (error bars, ribbons, gradients), composition (facets, patchwork, GridSpec), and tables (gt, kableExtra, sparklines). Workshop 3 covered 41 distinct chart types across these families.

### 2. Applying to Netflix
The Netflix dataset (8,807 titles: 6,131 movies + 2,676 TV shows) provides eight analytical questions: (a) genre dominance → sorted bar, (b) movie duration distribution → histogram, (c) rating by content type → stacked bar, (d) movie/TV split → waffle, (e) release trends → line, (f) duration by genre → ridgeline, (g) TV show seasons → boxplot, (h) country of origin → lollipop. Each chart was selected by following the decision flowchart.

### 3. The Complete Chart Vocabulary
After three workshops, students command 41 chart types: 8 distribution, 6 comparison, 5 relationship, 5 proportion, 4 ranking, 5 uncertainty, 4 composition, and 4 table types. Combined with the Grammar of Graphics (W02) and design principles (W01), this forms a complete visual communication toolkit.

### 4. Course Project 1: Visual Redesign Portfolio
Students select 5 published visualisations, critique each using Tufte's principles, redesign in R and Python, and write 300-word rationales. Due at end of Workshop 3 (10% of total grade).
