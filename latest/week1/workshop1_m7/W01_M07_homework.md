# Workshop 1 · Module 7 — Homework
## Critique & Redesign Portfolio

**Due**: Before Workshop 1, Module 9  
**Format**: PDF report (max 6 pages) + R script + Python script  
**Weight**: Part of Workshop 1 homework (5% of total grade)

---

### Part A — Four-Quadrant Critique (30 points)

1. Find **two** published charts from different sources (news, corporate
   reports, academic papers, social media dashboards).

2. For each chart, apply the four-quadrant critique framework:
   - **Data Integrity**: Compute the lie factor (if applicable), check axis
     labels, note any cherry-picking.
   - **Visual Encoding**: Identify the variable-to-channel mapping. Is the
     chart type appropriate for the analytical goal?
   - **Design Craft**: Assess data-ink ratio, typography, colour, whitespace.
   - **Communication**: Can you extract the takeaway in under 10 seconds?

3. Rate each quadrant on a 1–5 scale and write 100 words per chart.

4. Include screenshots or URLs of the original charts.

---

### Part B — Redesign Implementation (50 points)

1. Select the weaker of the two charts from Part A.

2. Execute the five-step redesign workflow:
   a. **Identify**: Write one sentence describing the core problem.
   b. **Diagnose**: Name the critique quadrant.
   c. **Sketch**: Include a hand-drawn or digital sketch (photo is fine).
   d. **Implement**: Produce the redesign in both R and Python using
      synthetic data that mimics the original structure.
   e. **Validate**: Write 100 words explaining how the redesign addresses
      each diagnosed problem, referencing the specific Modules 1–6 principle.

3. Present the before/after pair side by side.

---

### Part C — Peer Review Report (20 points)

1. Exchange your Part B redesign with a classmate.

2. Apply the five-step peer review protocol to their redesign.

3. Provide written feedback using the SBI model (Situation, Behaviour,
   Impact). Include at least one strength and one improvement suggestion.

4. Write 150 words summarising the feedback received on your own redesign.
