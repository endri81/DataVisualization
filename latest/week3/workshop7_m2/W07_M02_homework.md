# Workshop 7 · Module 2 — Homework
## Exploratory vs Explanatory Visualization

**Due**: Before Workshop 7, Module 4
**Format**: R script + Python script + PDF report (max 6 pages)
**Weight**: Part of Workshop 7 homework (5% of total grade)

### Part A — Dashboard vs Story Slide (30 points)
Choose one finding from your W06 work (Netflix or e-Car). Produce two versions of a chart that communicates this finding: (a) a **reader-driven dashboard** version (descriptive title, legend, all data, gridlines), (b) an **author-driven story slide** version (declarative title, grey+accent, direct labels, annotations). Produce in both R and Python. In 150 words, list each transformation you applied (use the 8-point checklist from the notes) and explain why each change serves the audience.

### Part B — Martini Glass Mockup (30 points)
Design a Martini Glass presentation for the Netflix dataset: (a) produce 3 guided slides (context → complication → resolution) as a vertical panel, (b) describe in 150 words what the "dashboard exploration" phase would contain (which filters, which metrics, which interactions). You do not need to build the dashboard — just describe it and sketch the layout.

### Part C — Spectrum Classification (20 points)
For each of the following outputs, classify it on the Segel & Heer spectrum (0% = fully reader-driven, 100% = fully author-driven) and justify in one sentence: (a) your W06-M10 lab dashboard, (b) a Quarto HTML report with interactive plotly charts, (c) a 10-slide board presentation, (d) a public Tableau dashboard for a city's open data portal, (e) a Jupyter notebook shared with your data science team.

### Part D — Reverse Transformation (20 points)
Take the author-driven story slide from Part A and transform it INTO a dashboard component. List the specific changes needed (restore legend, add filters, etc.). In 100 words, explain what information is gained and what narrative clarity is lost in this reverse transformation.
